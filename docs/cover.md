# md2docx

**Markdown to DOCX Converter**

**เครื่องมือแปลง Markdown เป็น DOCX**

---

**Version 0.1.0**

**เวอร์ชัน 0.1.0**

---

A Rust-based tool for converting Markdown documentation to professional Microsoft Word documents with full Thai language support.

เครื่องมือที่พัฒนาด้วยภาษา Rust สำหรับแปลงเอกสาร Markdown เป็นเอกสาร Microsoft Word ระดับมืออาชีพ พร้อมรองรับภาษาไทยอย่างสมบูรณ์
